Manual de Endpoints da API (Resumo)

Usuários

Criar usuário

POST /usuarios Body (JSON): { “nome”: “João”, “email”:
“email@email.com”, “senha”: “123456” }

Login

POST /usuarios/login Body (JSON): { “email”: “email@email.com”, “senha”:
“123456” } Retorno: token JWT

Listar todos os usuários

GET /usuarios

Buscar usuário por ID

GET /usuarios/:id

Atualizar usuário

PUT /usuarios/:id

Deletar usuário

DELETE /usuarios/:id

Projetos (Requer Token JWT)

Enviar no Header: Authorization: Bearer SEU_TOKEN_AQUI

Criar projeto

POST /projetos Body (JSON): { “titulo”: “Meu Projeto”, “descricao”:
“Alguma descrição” }

Listar projetos do usuário logado

GET /projetos

Buscar projeto por ID

GET /projetos/:id

Atualizar projeto

PUT /projetos/:id

Deletar projeto

DELETE /projetos/:id

Tarefas (Requer Token)

Mesmo padrão dos projetos.
